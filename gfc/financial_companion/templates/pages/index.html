About



Greggor will be you guide throughout this application
He'll be around to help you manage your spending and saving
You can always find him on the top hand corner of the page
He will change mood depending on how you interact with the site
This includes the date, time and how well you manage your financial accounts
To keep him happy, make sure to keep your financial life in order
Your goal should be to make sure he doesn't get upset.





Features



This is your guide to the features our product provides.
Upon creating an account and logging in you will be met with a dashboard which will give you a brief overview your accounts and transactions. If you have set up any bank accounts with an interest rate you will be able to view a graph of your saving projections.
At the top of the page you will see the navigation bar. This allows you to access all of the features of the website. Here is a description of each of these items:
Accounts, this will take you to a page where you are able to view all your different accounts. You are able to view further information regarding each account, add new accounts and delete any existing accounts
Transactions, the user is able to view their transactions. The user will see a table display all their transactions. They can filter these to see all sent or all receiving transactions. The user is also able to record new transactions. Users can also view any subscriptions or standing order they have setup to track by clicking the subscriptions dropdown options of Transactions. Again users are able to add a new standing order or subscription.
Categories, users can view all the categories they have created. These allow the user to log transactions of a particular type. Users are able to view more information about each category or delete any existing one. When view a category the user will see information regarding the category, any targets created for the target and any transactions linked to that category.
Groups, This is where users can create or join groups in order to compete with other in making you stay within spending targets and complete incoming transactions related targets. Owners of a group manage the group in terms of its members and is able to transfer their ownership if they so wish. Each group will also have a leaderboard where users can view their position amongst their fellow members.
Quiz, Users can take a quiz which will help them better their financial knowledge. They are able to take this quiz any number of times as well as their is no time limit for a quiz. The users previous results will be displayed so the users can track their progress.
Targets, There are three different types of targets users are to track. Account targets, category targets and user targets. Users will get notifications on appropriate pages to inform the users if they are nearing completion of these targets. There is also a page where the user can view all their targets.
Spending Summary, this is where the user will be able to keep track and view all of their progress regarding their spending. This page will include show different graphs show the user's spending
Finally Merch, anyone is welcome to purchase some products which include our financial companion Greggor if they would like to on our Etsy shop.
